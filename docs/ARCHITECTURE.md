## 核心模块说明

### 1. 前端模块

#### Dashboard (仪表盘)
- 展示统计数据：脚本总数、执行次数、通过率
- 执行趋势图表（近7天）
- 最近更新的脚本
- 最近生成的报告

#### Playground (可视化编辑器)
- Monaco Editor 编辑器（YAML 语法高亮）
- 实时 YAML 验证
- 预设模板选择
- 快速执行测试
- 执行日志展示

#### 脚本管理
- 脚本列表（支持搜索、筛选）
- 新建/编辑脚本
- YAML 导入/导出
- 脚本复制
- 单个脚本执行

#### 批量执行
- 多选脚本批量执行
- 并行/串行模式
- 并发数配置
- 出错时停止选项
- 实时进度展示

#### 报告中心
- 报告列表（支持筛选）
- 报告详情查看
- HTML 报告导出
- 失败原因分析

### 2. 后端模块

#### Script Service
- YAML 脚本的 CRUD 操作
- 文件存储管理
- 元数据管理
- 导入/导出处理

#### Execute Service
- 调用 Midscene CLI 执行脚本
- 实时进度推送 (SSE)
- 批量执行管理
- 结果解析和处理

#### Report Service
- 报告生成和存储
- HTML 报告渲染
- 统计数据计算
- 历史记录管理

### 3. 数据存储

```
data/
├── scripts-meta.json    # 脚本元数据
└── reports-meta.json    # 报告元数据

scripts/                  # YAML 脚本文件
├── script-xxx.yaml
└── ...

reports/                  # HTML 报告文件
├── batch-xxx.html
└── ...

temp/                     # 临时文件
```

## 安全考虑

1. **AI API Key 管理**：使用环境变量，不存储在前端
2. **文件上传限制**：限制 YAML 文件大小和类型
3. **脚本执行隔离**：在沙箱环境中执行测试脚本
4. **数据备份**：定期备份脚本和报告数据

## 性能优化

1. **前端优化**：
   - 路由懒加载
   - 组件按需加载
   - 数据缓存

2. **后端优化**：
   - 批量执行使用连接池
   - 报告生成使用队列
   - 静态文件使用 CDN

3. **执行优化**：
   - 并行执行支持
   - 资源复用
   - 智能重试机制
